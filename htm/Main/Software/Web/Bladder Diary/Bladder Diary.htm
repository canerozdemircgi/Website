Bladder Diary is a web application that reads a specific formatted text file in order to track inputs and outputs of human body fluid.
It is designed to demonstrate data in a clear way and present user-friendly interface for urology doctors and patients.
It generates statistics and shows informative charts which provides useful insights that could be beneficial for diagnosis and treatment.
It has been built as lightweight client-side solution with html, js, css without any framework or backend dependencies to ensure privacy, ease of use and compatibility among diverse operating systems.
